## App Peliculas
Aplicación web donde los usuarios puedan buscar películas, ver detalles, escribir y leer reseñas.
## Requisitos previos
- Node.js >= 22
- npm o Yarn

## Instalación

1. Clona este repositorio en tu máquina local:
```bash
git clone https://github.com/angelvcuenca/front-movies-libelula.git
```
2. Navega hasta el directorio de tu proyecto:
```bash
cd front-movies-libelula
```
3. Instala las dependencias del proyecto

```bash
npm install
# O con Yarn
yarn install

```


## Desarrollo

Para ejecutar el proyecto en modo de desarrollo:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Uso

Para usar el proyecto en modo de desarrollo, ingresa al siguiente url en tu local:
```bash
http://localhost:3000
```
